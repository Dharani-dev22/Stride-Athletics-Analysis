## 📝 Project Overview
This project demonstrates an end-to-end data analysis pipeline for "Stride Athletics," a fictional athletic retail brand. The objective was to create a centralized executive dashboard to monitor Year-to-Date (YTD) business performance across key metrics: Sales, Profit, and Order Volume.

The dashboard allows leadership to gain immediate visibility into current performance against prior-year benchmarks, segmented by sales channel (Online vs. Retail).

## 🛠 Tools & Technologies Used
* **Python (Pandas/Numpy):** Used for synthetic data generation, date handling, and financial metric simulation.
* **Tableau Public:** Used for interactive dashboard design, implementing dual-axis charts, and creating executive-level KPI cards.
* **CSV:** Used for data storage and seamless integration between the Python pipeline and Tableau.

## 💡 Key Business Insights
Through the visual analysis, three major trends were identified:
1. **Sales Performance:** Comparison of current year revenue against prior year benchmarks using synchronized dual-axis bullet charts.
2. **Profitability Trends:** High-level overview of year-over-year profitability, identifying the most efficient sales channels.
3. **Order Volume:** Monitoring transactional growth and seasonal order fluctuations across 2026.
